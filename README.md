# Python Module 00
Python Module 00 is the first Python module at 42 school. Here you learn basic concepts about working with methods and an introduction to OOP.

## Exercices
`ft_hello_garden.py (ex0)`: Prints a simple welcome message for the community garden.<br>
`ft_plot_area.py (ex1)`: Asks for length and width, then calculates and prints the plot area.<br>
`ft_harvest_total.py (ex2)`: Requests three harvest weights and prints their total.<br>
`ft_plant_age.py (ex3)`: Checks if a plant is ready to harvest based on its age in days.<br>
`ft_water_reminder.py (ex4)`: Determines whether plants need watering based on days since last watering.<br>
`ft_count_harvest_iterative.py (ex5)`: Counts from 1 to a given number using a loop, then prints “Harvest time!”.<br>
`ft_count_harvest_recursive.py (ex5)`: Same as the iterative version, but implemented with recursion.<br>
`ft_garden_summary.py (ex6)`: Collects garden name and plant count, then prints a formatted summary.<br>
`ft_seed_inventory.py (ex7)`: Displays seed inventory information depending on the unit type (packets, grams and area).<br>
`main.py`: A Python Module 00 tester provided by 42
